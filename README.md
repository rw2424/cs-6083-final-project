# Cookzilla

Cookzilla is a cooking and recipe website.

## Prerequisite

- Node.js: `v19.2.0` (other versions may also work, but I only tested this version).

## Quickstart

1. Install the dependencies:
   ```shell
   npm install
   ```
2. Run the development server:
   ```shell
   npm run dev
   ```
